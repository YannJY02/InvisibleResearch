# Supervisor Meeting Reports

This section stores version-controlled reports prepared for recurring
supervisor or research-group meetings. Name each report
`YYYY-MM-DD-short-topic.md` and add it to the index below.

## Boundaries

- Link to canonical material under [`research/`](../research/README.md) or
  [`papers/`](../papers/README.md) instead of copying source artifacts.
- Commit only privacy-safe summaries intended for repository collaborators.
  Keep raw messages and private source material in the local-only
  [`inbox/`](../inbox/README.md).
- These reports are communication records. Their presence does not grant
  **Paper Analysis** status or create a **Designation Event**; those authority
  rules remain defined in [`CONTEXT.md`](../CONTEXT.md).
